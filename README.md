La Nevera de Crisp — versión definitiva actual

Qué incluye:
- Home con la estética exacta de la nevera de referencia.
- Pantalla inicial formada solo por la nevera, sin cabecera, sin pie y sin menú externo.

Subida:
1. Descomprime este ZIP.
2. Sube todo el contenido a la raíz del repositorio, sustituyendo lo anterior.
3. Haz commit.
4. Recarga la página. Si el navegador seguía guardando algo, esta versión limpia el caché antiguo al abrirse.
